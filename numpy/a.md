# 矩阵

#### 1. 创建数组

###### 1.1 np.arange|np.array用法
```python
	import numpy as np6
	a = np.array([1,2,3,4,5])
	b = np.array(range(1, 6))
	c = np.arange(1, 6)
	np.arange|np.array用法(1, 6)
```

+ <font color=red >注意: 上面的a, b, c 内容相同, 注意arange和range的区别</font>
+ np.arange的用法: arange([start,] stop[, stop,], dtype=None)
